Descrição do Projeto – Sistema Web para Venda de Semijoias - Econjunto c/ João Tinoco

Este projeto consiste no desenvolvimento de um sistema web para venda de semijoias, criado utilizando HTML como base estrutural, com foco em organização, usabilidade e experiência do usuário. O sistema foi idealizado para atender tanto clientes finais quanto a gestão do catálogo de produtos, simulando o funcionamento de um e-commerce moderno.

A aplicação conta com uma interface intuitiva e responsiva, permitindo a navegação por diversas categorias de semijoias, como anéis, colares, brincos, pulseiras e kits promocionais. Cada produto possui uma página detalhada contendo imagem, descrição, preço e opção de compra.

Entre as principais funcionalidades do sistema, destacam-se:

Exibição organizada dos produtos por categorias

Visualização detalhada dos itens

Carrinho de compras para adição, remoção e atualização de produtos

Simulação do processo de compra

Layout responsivo, adaptado para desktop e dispositivos móveis

O projeto tem como objetivo aplicar conceitos fundamentais de desenvolvimento web, como estruturação semântica do HTML, boas práticas de organização de páginas e navegação clara, servindo como base para futuras evoluções, como integração com CSS, JavaScript, banco de dados e sistemas de pagamento.

Trata-se de uma solução que demonstra, de forma prática, o funcionamento de um sistema de e-commerce, voltado ao segmento de semijoias, unindo estética, funcionalidade e organização da informação.
